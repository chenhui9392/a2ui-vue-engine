# A2UI Vue Engine 统一样式规范

## 一、概述

本文档定义了 A2UI Vue Engine 渲染器的统一样式标准，旨在保证页面美观性、一致性和可维护性。

**核心设计理念**：通过定义 Card 宽度标准来控制整体表单布局，输入组件宽度自适应填充 Card，实现统一美观的效果。

## 二、颜色系统

|---------|------|---------|

| `--a2-color-primary-light` | `#79bbff` | 主色悬停状态 |
| `--a2-color-primary-dark` | `#337ecc` | 主色按下状态 |

| 颜色变量 | 色值 | 使用场景 |
|---------|------|---------|
| `--a2-color-success` | `#67c23a` | 成功提示、完成状态 |

| `--a2-color-danger` | `#f56c6c` | 错误提示、删除按钮 |

### 2.3 按钮颜色规范

根据按钮文本内容自动匹配颜色：

| 文本关键词 | 按钮类型 | 颜色 |
|-----------|---------|------|
| 提交、确认、保存、确定 | primary | 蓝色 (#409eff) |
| 取消、关闭、返回 | default | 灰色边框 |
| 删除、移除 | danger | 红色 (#f56c6c) |

## 三、间距系统

### 3.1 基础间距单位

以 **4px** 为基准单位：

| 变量名 | 值 | 使用场景 |

| `--a2-spacing-xs` | 4px | 最小间距、图标与文字间 |

| `--a2-spacing-md` | 12px | 中等间距、表单项间距 |

| `--a2-spacing-xxl` | 24px | 最大间距、页面区块间距 |

| 表单项之间 | 12px | `--a2-gap-block` |
| 页面区块之间 | 24px | `--a2-gap-section` |

## 四、Card 宽度标准（核心）

### 4.1 Card 宽度定义

**通过 Card 宽度控制整体表单布局，输入组件自动填充**

| 尺寸 | 宽度 | 变量 | 使用场景 |
|-----|------|------|---------|
| sm (小) | 400px | `--a2-card-width-sm` | 简单表单、筛选面板 |
| md (默认) | **560px** | `--a2-card-width-md` | 标准表单、请假单、申请单 |
| lg (大) | 720px | `--a2-card-width-lg` | 复杂表单、详情页 |
| xl (超大) | 960px | `--a2-card-width-xl` | 大表单、报表 |
| full (全宽) | 100% | `--a2-card-width-full` | 占满容器宽度 |

### 4.2 设计原理

```
┌─────────────────────────────────────────┐
│  Card (width: 560px)                     │
│  ┌─────────────────────────────────────┐│
│  │ 标签 (100px)    输入框 (100%自适应) ││
│  │ 标签            下拉框 (100%自适应) ││
│  │ 标签            日期框 (100%自适应) ││
│  │ 标签            Textarea (100%自适应││
│  └─────────────────────────────────────┘│
│                      [提交] [取消]       │
└─────────────────────────────────────────┘
```

**要点：**
- Card 定义整体宽度
- 输入组件宽度 100%（自适应填充）
- 标签宽度统一 100px
- 同一 Card 内所有输入框自动对齐

## 七、布局规范

### 7.1 表单布局

```
┌─────────────────────────────────────────┐
│  标签 (100px)    输入框 (填充 Card)      │  ← 间距 12px
│                                         │
│  标签            下拉框 (填充 Card)      │  ← 间距 12px
│                                         │
│  标签            日期选择器 (填充 Card)  │  ← 间距 12px
│                                         │
│  标签            Textarea (填充 Card)   │  ← 间距 12px
│                                         │
│                    [提交] [取消]         │  ← 按钮间距 10px
└─────────────────────────────────────────┘
```

**规范要点：**
- Card 宽度决定整体布局
- 标签宽度：统一 100px
- 输入组件宽度：100%（自适应）
- 表单项间距：12px
- 操作按钮组间距：10px
- 操作按钮组靠右对齐

## 八、组件使用指南

### 8.1 A2Card 使用示例（控制表单宽度）

```vue
<!-- 默认表单宽度 560px -->
<A2Card>
  <!-- 表单内容 -->
</A2Card>

<!-- 简单表单 400px -->
<A2Card width="sm">
  <!-- 简单表单内容 -->
</A2Card>

<!-- 复杂表单 720px -->
<A2Card width="lg">
  <!-- 复杂表单内容 -->
</A2Card>

<!-- 大表单 960px -->
<A2Card width="xl">
  <!-- 大表单内容 -->
</A2Card>
```

### 8.2 输入组件（自动填充 Card 宽度）

```vue
<!-- 输入框：宽度 100% 自动填充 -->
<A2Input placeholder="请输入" />
<A2Select :options="options" placeholder="请选择" />
<A2DatePicker placeholder="请选择日期" />
```

### 8.3 A2TextField（带标签）

```vue
<!-- 普通 text 输入 -->
<A2TextField label="姓名" placeholder="请输入姓名" />

<!-- textarea -->
<A2TextField label="备注" type="textarea" rows="4" />
```

### 8.4 A2Button

```vue
<!-- 按钮文字决定颜色 -->
<A2Button text="提交" />  <!-- 自动蓝色 -->
<A2Button text="取消" />  <!-- 自动灰色 -->
<A2Button text="删除" />  <!-- 自动红色 -->
```

## 九、最佳实践总结

### 推荐做法

1. **通过 Card width 控制表单整体宽度**
2. **简单表单**：使用 `width="sm"` (400px)
3. **标准表单**：使用默认 `width="md"` (560px)
4. **复杂表单**：使用 `width="lg"` (720px)
5. **同一 Card 内所有输入框自动对齐**

### 禁止行为

1. 禁止手动指定输入组件宽度（请通过 Card width 控制）
2. 禁止在同一 Card 内使用不一致的间距
