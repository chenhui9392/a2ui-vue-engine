# Components Overview

A2UI provides a comprehensive set of JSON-driven components.

## Categories

### Layout Components

- [A2Card](/en/components/a2-card) - Card container with width variants
- [A2Row](/en/components/a2-row) - Horizontal row layout
- [A2Column](/en/components/a2-column) - Vertical column layout
- [A2List](/en/components/a2-list) - Repeating item list

### Form Components

- [A2TextField](/en/components/a2-text-field) - Form field with label
- [A2Input](/en/components/a2-input) - Basic input field
- [A2Select](/en/components/a2-select) - Basic select component
- [A2SelectField](/en/components/a2-select-field) - Select dropdown (with label)
- [A2DatePicker](/en/components/a2-date-picker) - Date selection
- [A2DateTimeInput](/en/components/a2-date-time-input) - Date and time selection
- [A2ChoicePicker](/en/components/a2-choice-picker) - Multi/single choice picker

### Display Components

- [A2Text](/en/components/a2-text) - Text display with variants
- [A2Icon](/en/components/a2-icon) - Icon element
- [A2InfoField](/en/components/a2-info-field) - Info display field (label + value)

### Action Components

- [A2Button](/en/components/a2-button) - Button with actions
