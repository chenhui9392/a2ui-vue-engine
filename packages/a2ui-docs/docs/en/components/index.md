# Components Overview

A2UI provides a comprehensive set of JSON-driven components.

## Categories

### Layout Components

- [A2Card](/components/a2-card) - Card container with width variants
- [A2Row](/components/a2-row) - Horizontal row layout
- [A2Column](/components/a2-column) - Vertical column layout
- [A2List](/components/a2-list) - Repeating item list

### Form Components

- [A2TextField](/components/a2-text-field) - Form field with label
- [A2Input](/components/a2-input) - Basic input field
- [A2SelectField](/components/a2-select-field) - Select dropdown
- [A2DatePicker](/components/a2-date-picker) - Date selection
- [A2DateTimeInput](/components/a2-date-time-input) - Date and time selection

### Display Components

- [A2Text](/components/a2-text) - Text display with variants
- [A2Icon](/components/a2-icon) - Icon element

### Action Components

- [A2Button](/components/a2-button) - Button with actions
