# 组件概览

A2UI 提供完整的 JSON 驱动组件库。

## 分类

### 布局组件

- [A2Card](/zh/components/a2-card) - 卡片容器，多种宽度规格
- [A2Row](/zh/components/a2-row) - 水平行布局
- [A2Column](/zh/components/a2-column) - 垂直列布局
- [A2List](/zh/components/a2-list) - 重复项列表

### 表单组件

- [A2TextField](/zh/components/a2-text-field) - 带标签的表单字段
- [A2Input](/zh/components/a2-input) - 基础输入框
- [A2SelectField](/zh/components/a2-select-field) - 下拉选择框
- [A2DatePicker](/zh/components/a2-date-picker) - 日期选择
- [A2DateTimeInput](/zh/components/a2-date-time-input) - 日期时间选择

### 展示组件

- [A2Text](/zh/components/a2-text) - 文本展示，多种样式
- [A2Icon](/zh/components/a2-icon) - 图标元素

### 操作组件

- [A2Button](/zh/components/a2-button) - 按钮，支持动作
